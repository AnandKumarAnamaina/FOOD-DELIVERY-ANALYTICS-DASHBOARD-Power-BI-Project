# 🍔 Food Delivery Analytics Dashboard

## 📌 Project Overview

This Power BI project analyzes food delivery business data and provides insights into revenue, delivery performance, and restaurant operations.

The dashboard helps understand:

✔ Revenue Performance

✔ Order Analysis

✔ Delivery Efficiency

✔ Restaurant Comparison

✔ Delivery Status Monitoring

## 📊 Dashboard Features

* Total Revenue KPI
* Order Count KPI
* Average Delivery Time KPI
* Revenue by Restaurant
* Delivery Time Trend Analysis
* Delivery Status Distribution
* Restaurant-wise Order Analysis
* Interactive Filters

## 🛠 Tools Used

* Power BI
* Power Query
* DAX
* Data Modeling
* Dashboard Design

## 📈 Key Business Insights

* Restaurant revenue comparison
* Fast vs Delayed delivery monitoring
* Delivery performance tracking
* Order volume analysis
* Interactive business reporting
